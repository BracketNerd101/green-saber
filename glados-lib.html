Code Library for a team of Developers

Every developer on our team uses the UI Kit to produce our React JS applications. It contains all of the custom design elements needed to build our projects successfully. The internal application contains code snippits, design annotations, theme switcher and a sidebar navigation.

Our theme switcher allows us to seamlessly swap css files to match any of the 12 florida blue clients. All of these essential parts help the developers focus on the component and speed up their dev workflow.

I worked with my manager, Eric Crawford; Kim Olmoguez and a team of developers to create the application.
