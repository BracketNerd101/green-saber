Code Library for a team of Developers

Developers used the UI Kit to produce React JS applications. The comprehensive approach contains custom design elements needed to build each project successfully. The internal application contains code snippets, design annotations, a theme switcher, and sidebar navigation.

The theme switcher allows seamless a swap of CSS files to match any of the 12 Florida blue clients. All essential tools in helping developers focus on the component while expediting the dev workflow.

I worked with my manager, Eric Crawford; Kim Olmoguez, and a team of developers to create the application.
